1. Frontend Setup

Create Project:
  • New Method: npm create vite@latest [file name] -- --template react
  • Old Method: npm create vite@latest [file name]
Navigation:
  • cd [file name]
Start Dev:
  • npm run dev

2. CSS Frameworks

Tailwind CSS:
  • npm install tailwindcss @tailwindcss/vite
  • vite.config.ts setup:

3rd line -> import tailwindcss from '@tailwindcss/vite | 2nd para 2nd line plugins array 2nd -> tailwindcss()
  • index.css: Add @import "tailwindcss"; at the top.
Bootstrap (HTML Integration):

  • Add CDN link in index.html <head>:
  • <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

  • Add CDN link in index.html <body> (bottom):
  • <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

3. Dependencies & Libraries

Install Command:
  • npm install axios react-router-dom react-toastify @reduxjs/toolkit react-redux face-api.js
Routing:
    In main.jsx:
  • import { BrowserRouter } from 'react-router-dom';
TensorFlow (Face-API):
  • Repo: https://github.com/justadudewhohacks/face-api.js/tree/master
  • npm i face-api.js
Gemini API:
  • https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent

4. Backend Development

Init:
  • npm init -y
File Rename:
    Rename index.js to main.js
Dependencies:
  • npm i express mongoose dotenv cors bcryptjs jsonwebtoken cookie-parser nodemailer express-rate-limit
Dev Tool:
  • npm i -D nodemon
Scripts (package.json):
  • "start": "node main.js"
  • "dev": "nodemon --watch main.js"
Start Backend:
  • ** Dev Mode: npm run dev
  • **Production Start: npm run dev
    • // command terminal "ctrl + c" server stop

MongoDB Community:

  • mongodb://localhost:27017/[your_database_name] or mongodb://127.0.0.1:27017/[sample_db]
Atlas (Cloud):
  • https://www.mongodb.com/products/platform/atlas-database
Atlas Connection URI:
  • mongodb+srv://[db_user]:[password]@b43-db.o7geprk.mongodb.net/?appName=[app]
Network Access:
  • Set entry to 0.0.0.0/0
Testing (Postman & Thunder Client):
  • Postman.com (Desktop App) / Thunder Client (VS Code Extension)
